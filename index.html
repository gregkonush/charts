Installation Instructions

1. Add the Helm repository:

helm repo add proompteng https://charts.proompteng.ai

2. Update your local Helm chart repository cache:

helm repo update

3. Install the chart (e.g., affine):

helm install affine proompteng/affine

Replace affine with your desired release name.

Available Helm Charts

affine

AFFiNE is an open-source, privacy-first knowledge base merging docs, whiteboards & databases. An alternative to Notion & Miro.
